## Description

`基于 RuoYi / RuoYi-Vue 二次开发的仓储与财务一体化管理系统`

## About

仓链管理系统是一套基于 `RuoYi` / `RuoYi-Vue` 体系进行二次开发的前后端分离业务系统。

系统保留了 RuoYi 原有的用户权限、角色菜单、日志监控、定时任务、代码生成等基础平台能力，并在此基础上扩展了以下业务方向：

- 仓库管理
- 库位管理
- 入库管理
- 出库管理
- 库存管理
- 调拨管理
- 盘点管理
- 客户管理
- 供应商管理
- 产品管理
- 应收管理
- 应付管理
- 收款管理
- 付款管理
- 费用管理
- 资金账户管理
- 核销管理
- 财务报表管理

适合作为基于 RuoYi 框架继续进行仓储、财务、经营业务扩展的项目基础。
