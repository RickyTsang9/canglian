# 2026-04-22 更新日记

## 对比基线

- 上个版本提交：`f5aefa2`
- 上个版本说明：`feat: deliver jdy light erp upgrade`
- 本次版本定位：在精斗云轻量进销存底座上，继续补齐金蝶云星辰商贸标准版/专业版的关键闭环能力

## 本次版本结论

本次更新将系统从“订单驱动轻量 ERP”推进到“星辰式商贸经营闭环”的第一阶段。系统现在具备凭证生成、凭证回写、凭证冲销、基础总账接口、发票与往来联动、经营分析报表等能力，适合作为小微商贸、批发、多仓、多门店场景的轻量替代方案继续上线验证。

## 业务能力升级

### 1. 业财税闭环增强

- 凭证事件从“预留记录”升级为可操作的凭证生命周期
- 新增凭证生成、凭证回写、凭证冲销能力
- 凭证事件补齐凭证号、凭证状态、凭证日期、回写状态、冲销凭证号等字段
- 发票确认后自动同步应收或应付候选记录
- 销售发票联动客户应收，采购发票联动供应商应付

### 2. 基础总账接口

新增轻量总账接口层，先做业务财务衔接，不一次性建设完整总账：

- 会计科目
- 辅助核算
- 会计期间
- 期间结账
- 期间反结账

### 3. 经营分析报表

新增经营视角报表入口：

- 销售毛利
- 客户贡献
- 商品周转
- 库存余额
- 资金分析

报表 SQL 已避免一对多关联导致的金额、库存重复统计问题。

## 前端升级

- 凭证事件页面新增凭证号、凭证状态、冲销凭证号展示
- 凭证事件页面新增“生成”“回写”“冲销”操作
- 新增基础总账接口页面，支持会计科目、辅助核算、会计期间查看和期间状态处理
- 新增经营分析报表页面，支持按日期区间查询销售毛利、客户贡献、商品周转、库存余额、资金分析

## 后端升级

- 新增 `FinGeneralLedgerController`
- 新增 `IFinGeneralLedgerService` 与实现类
- 新增 `FinGeneralLedgerMapper` 与 MyBatis XML
- 扩展 `FinVoucherEvent` 字段与服务方法
- 扩展 `FinInvoiceServiceImpl`，发票确认时同步往来单据
- 扩展 `FinReportController`、`IFinReportService`、`FinReportMapper`，提供经营分析接口
- 保持 `controller -> service -> mapper` 分层方式
